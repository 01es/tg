\section{A Short Introduction to Apache Maven}
  
  Apache Maven is a software project management and comprehension tool. 
  It is based on the concept of a \emph{project object model} or simpy POM. 
  Maven can manage a project's dependencies and build from a central piece of information declared in its POM file.

  \lstset{language=XML,
	  escapechar=\%,
	  morekeywords={project, modelVersion, groupId, artifactId, packaging, packaging, version, name, description},
	  numbers=left, numberstyle=\tiny, basicstyle=\scriptsize\color{basiccolor}, stepnumber=1, numbersep=5pt, keywordstyle=\bfseries\color{codefgcolor}, stringstyle=\color{stringcolor}}
